This is a two level script set-up: a lunching script fastp_quality_control_lunch.sh that lunches the corresponding slurm batch script depending on the provided parameters.

Positional parameters:
1) PROYECT(1st parameter): name of the folder dataset
2) TYPE(2nd parameter): type of FASTQ files, PAIRED(PE) or SINGLE(SE)
3) PG(3rd parameter): indicate if treat PolyG tail(Y) or not(N)
4) RAW(4th parameter): indicate type of raw input directory, Downloaded_FASTQs(D) or Treated Raw_FASTQs(T)

Example:

Basic usage:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT TYPE PG RAW
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lunch Fastp for PAIRED and raw input dir in Downloaded_FASTQs folder:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT PE N D
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lunch Fastp for PAIRED PolyG and raw input dir in Downloaded_FASTQs folder:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT PE Y D
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lunch Fastp for SINGLE and raw input dir in Downloaded_FASTQs folder:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT SE N D
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lunch Fastp for SINGLE PolyG and raw input dir in Downloaded_FASTQs folder:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT SE Y D
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lunch Fastp for PAIRED and raw input dir in Raw_FASTQs folder:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT PE N T
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lunch Fastp for PAIRED PolyG and raw input dir in Raw_FASTQs folder:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT PE Y T
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lunch Fastp for SINGLE and raw input dir in Raw_FASTQs folder:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT SE N T
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lunch Fastp for SINGLE PolyG and raw input dir in Raw_FASTQs folder:
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
bash fastp_quality_control_lunch.sh PROYECT SE Y T
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Comments:

- All scripts need to be in the same folder.

- By default, the DATASETS_DIR=/storage/tbc/gut_core/CoreV2/Muestras_SPE. If you have other path for your datasets change the variable in the fastp_quality_control_lunch.sh script.

- By default, the LOG_DIR=/home/sapies/logs/QC/fastp. If you have other path for your datasets change the LOG_DIR variable in the fastp_quality_control_lunch.sh script.

- By default, the batch scripts are set to --qos=medium, --cpus-per-task=8, --mem=10gb, --time=7-00:00:00 . If you have memory or time problems change the parameters accordingly in the batch scripts.
